### 五、教学过程

#### 第一环节：导入（15分钟）

**教学内容**
- 引入问题：如何衡量业务的好坏？
  - 没有指标，就无法衡量
  - 无法衡量，就无法管理

- 指标的作用
  - 描述业务现状
  - 评估业务效果
  - 指导业务决策

**教学活动**
- 教师提问：你如何衡量一个电商平台的好坏？
- 学生思考：用什么指标来衡量？

---

#### 第二环节：指标的概念与原则（40分钟）

**教学内容**
- 什么是指标
  - 指标是衡量业务状况的数值
  - 指标是业务目标的量化表示

- 好的指标的标准（SMART原则）

| 原则 | 英文 | 含义 | 示例 |
| :--- | :--- | :--- | :--- |
| **具体** | Specific | 指标要明确 | "用户数"比"用户活跃度"更具体 |
| **可衡量** | Measurable | 结果要可量化 | 销售额、订单数 |
| **可实现** | Achievable | 目标要现实 | 不要设定不切实际的目标 |
| **相关** | Relevant | 与业务相关 | 指标要与业务目标相关 |
| **有时限** | Time-bound | 有明确的时间要求 | 日活、月活、年收入 |

- 指标的分类
  - 结果指标：衡量最终结果（如销售额、利润）
  - 过程指标：衡量过程表现（如转化率、留存率）
  - 先行指标：预测未来趋势（如用户注册数、浏览量）

**教学活动**
- 教师讲解指标的概念与原则
- 学生用SMART原则评估指标

---

#### 第三环节：指标定义方法（50分钟）

**教学内容**
- 北极星指标
  - 定义：最能反映业务核心价值的单一指标
  - 特点：与业务目标强相关、可衡量、可行动
  - 示例：
    - 电商：GMV（成交总额）
    - 社交：DAU（日活跃用户数）
    - 内容：用户时长

- OSM模型（目标-策略-指标）

| 层次 | 定义 | 示例 |
| :--- | :--- | :--- |
| **目标（Objective）** | 业务目标 | 提升销售额 |
| **策略（Strategy）** | 实现目标的策略 | 提升转化率、提高客单价 |
| **指标（Measure）** | 衡量策略的指标 | 转化率、客单价、复购率 |

- 指标分层体系
  - L1：公司级指标（北极星指标）
  - L2：部门级指标
  - L3：业务线指标
  - L4：功能级指标

- 指标字典的建立
  - 指标名称
  - 指标定义
  - 计算公式
  - 数据来源
  - 负责人
  - 更新频率

**教学活动**
- 教师讲解指标定义方法
- 学生用OSM模型设计指标

---

#### 第四环节：常见业务指标（30分钟）

**教学内容**
- 电商类指标

| 指标 | 定义 | 计算公式 |
| :--- | :--- | :--- |
| **GMV** | 成交总额 | 所有订单金额之和 |
| **转化率** | 下单用户占访问用户的比例 | 下单用户数 / 访问用户数 |
| **客单价** | 平均每单金额 | GMV / 订单数 |
| **复购率** | 重复购买用户的比例 | 重复购买用户数 / 总购买用户数 |

- 用户类指标

| 指标 | 定义 | 计算公式 |
| :--- | :--- | :--- |
| **DAU** | 日活跃用户数 | 当日登录或使用产品的用户数 |
| **MAU** | 月活跃用户数 | 当月登录或使用产品的用户数 |
| **留存率** | 用户在一定时间后仍活跃的比例 | 第N天仍活跃的用户数 / 第1天新增用户数 |
| **用户时长** | 用户使用产品的平均时长 | 总使用时长 / 用户数 |

- 案例：价格指数的指标定义
  - 价格指数：衡量价格变动的指标
  - 计算方法：加权平均
  - 数据来源：电商平台价格数据

**教学活动**
- 教师讲解常见业务指标
- 学生理解指标的定义与计算

---

#### 第五环节：课堂研讨（15分钟）

**研讨议题一：指标设计练习**
- 讨论题：假设你负责一个在线教育平台，请设计：
  1. 一个北极星指标
  2. 用OSM模型设计3个关键指标
- 目的：练习指标设计

**研讨议题二：指标体系评审**
- 讨论题：各组汇报指标设计后，其他组提出问题和建议：
  1. 指标是否符合SMART原则？
  2. 指标是否与业务目标相关？
  3. 指标是否有改进空间？
- 目的：培养指标评审能力

**教学活动**
- 学生分组讨论
- 各组汇报指标设计
- 教师点评与总结

---

#### 第六环节：课堂小结（10分钟）

**小结内容**
- 本讲介绍了指标的概念、原则与定义方法
- 强调了SMART原则和OSM模型的重要性
- 介绍了常见业务指标的定义与计算
- 指标是数据分析的基础，好的指标设计是成功的一半
