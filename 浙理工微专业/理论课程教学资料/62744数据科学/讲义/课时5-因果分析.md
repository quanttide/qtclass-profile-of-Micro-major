# 第5章 因果分析

## 5.1 章节概述

**章节摘要**
本章将介绍因果关系与相关关系的区别，讲解因果分析的基本方法。因果分析是数据科学的高级应用，掌握因果推断方法是数据科学家的核心竞争力。通过本章学习，学生将理解因果推断的基本框架。

**学习目标**

## 5.2 因果关系概述

### 5.2.1 相关不等于因果

**案例**：冰淇淋销量与溺水事故呈正相关

**问题**：吃冰淇淋会导致溺水吗？

**答案**：不是，是因为夏天（混淆变量）

### 5.2.2 因果关系的定义

- **原因**：导致结果发生的变量
- **结果**：被原因影响的变量
- **因果关系**：原因导致结果变化

### 5.2.3 相关关系的类型

- **正相关**：两个变量一起增加
- **负相关**：一个变量增加，另一个减少
- **不相关**：两个变量没有关系

### 5.2.4 因果推断的基本框架

- **混淆变量**：同时影响原因和结果的变量
- **中介变量**：原因通过它影响结果
- **调节变量**：影响因果关系的强度

### 5.2.5 因果推断的挑战

- 混淆变量的存在
- 反向因果
- 选择偏差

## 5.3 因果分析方法

### 5.3.1 随机对照试验（A/B测试）

**原理**：随机将样本分为实验组和对照组

**优点**：消除混淆变量的影响

**缺点**：成本高、可能不道德

**示例**：测试新广告的效果

### 5.3.2 准实验设计

**原理**：在无法进行随机对照试验时，使用自然实验

**方法**：断点回归、双重差分、工具变量

**优点**：成本低、可行性强

**缺点**：假设条件较多

### 5.3.3 观察性研究的因果推断

**原理**：使用观察性数据进行因果推断

**方法**：倾向得分匹配、回归控制

**优点**：数据易得

**缺点**：无法完全消除混淆变量

### 5.3.4 方法选择策略

| 场景 | 推荐方法 | 优缺点 |
| :--- | :--- | :--- |
| **可以随机分组** | A/B测试 | 最可靠，但成本高 |
| **有自然实验** | 准实验设计 | 成本低，但假设多 |
| **只有观察性数据** | 观察性研究 | 数据易得，但可靠性低 |

## 5.4 因果关系图

### 5.4.1 有向无环图（DAG）

**定义**：用有向边表示因果关系的图

**节点**：变量

**有向边**：因果关系

### 5.4.2 因果图的绘制方法

1. 识别所有相关变量
2. 确定变量之间的因果关系
3. 用有向边连接变量

### 5.4.3 因果图的解读

- **直接因果**：A → B
- **间接因果**：A → C → B
- **混淆变量**：A ← C → B
- **中介变量**：A → C → B

### 5.4.4 案例：冰淇淋销量与溺水事故

- **变量**：冰淇淋销量、溺水事故、夏天
- **因果关系**：夏天 → 冰淇淋销量，夏天 → 溺水事故
- **结论**：冰淇淋销量与溺水事故是相关关系，不是因果关系

## 5.5 案例分析：价格变动的因果分析

### 5.5.1 问题定义

- **研究问题**：价格变动是否会影响销量？
- **挑战**：价格和销量可能同时受其他因素影响

### 5.5.2 混淆变量识别

- 季节
- 促销活动
- 竞争对手价格

### 5.5.3 因果图绘制

```
季节 → 价格
季节 → 销量
促销 → 价格
促销 → 销量
价格 → 销量
```

### 5.5.4 因果分析方案

- 使用A/B测试：随机调整部分商品的价格
- 使用准实验设计：利用促销活动的自然实验
