# 第3章 统计

## 3.1 章节概述

**章节摘要**
本章将介绍描述性统计和推断统计的核心方法。统计是数据分析的基础，掌握统计方法是数据科学家的基本功。通过本章学习，学生将能够运用统计方法分析数据特征。

**学习目标**
完成本章学习后，读者应能够：
1.  **应用**描述性统计分析数据特征；
2.  **理解**推断统计的基本概念；
3.  **执行**基本的假设检验。

## 3.2 描述性统计

### 3.2.1 集中趋势

| 指标 | 定义 | 计算方法 | 适用场景 |
| :--- | :--- | :--- | :--- |
| **均值** | 所有数据的平均值 | 总和 / 个数 | 数据分布均匀时 |
| **中位数** | 排序后位于中间的值 | 排序后取中间值 | 数据有极端值时 |
| **众数** | 出现次数最多的值 | 统计频次 | 分类数据时 |

### 3.2.2 离散程度

| 指标 | 定义 | 计算方法 | 适用场景 |
| :--- | :--- | :--- | :--- |
| **方差** | 数据与均值的偏离程度 | 各数据与均值差的平方的平均 | 衡量数据波动 |
| **标准差** | 方差的平方根 | 方差开方 | 与数据同单位 |
| **极差** | 最大值与最小值的差 | 最大值 - 最小值 | 简单衡量波动 |

### 3.2.3 分布形态

| 指标 | 定义 | 含义 |
| :--- | :--- | :--- |
| **偏度** | 分布的对称性 | 正偏、负偏、对称 |
| **峰度** | 分布的尖锐程度 | 尖峰、平峰、正态 |

### 3.2.4 数据可视化

| 图表类型 | 适用场景 | 展示内容 |
| :--- | :--- | :--- |
| **直方图** | 展示数据分布 | 频次分布 |
| **箱线图** | 展示数据分散 | 中位数、四分位数、异常值 |
| **散点图** | 展示两个变量的关系 | 相关性 |

## 3.3 推断统计基础

### 3.3.1 总体与样本

- **总体**：研究对象的全部
- **样本**：从总体中抽取的部分
- **目的**：用样本推断总体

### 3.3.2 抽样方法

| 方法 | 定义 | 特点 |
| :--- | :--- | :--- |
| **简单随机抽样** | 每个个体被抽中的概率相等 | 简单、公平 |
| **分层抽样** | 按特征分层后抽样 | 保证各层都有代表性 |
| **整群抽样** | 随机抽取若干群组 | 适用于群组差异小的情况 |

### 3.3.3 置信区间

- **定义**：总体参数可能落入的区间
- **计算方法**：样本统计量 ± 误差范围
- **置信水平**：95%、99%等

---

## 3.4 假设检验

### 3.4.1 基本概念

- **原假设（H0）**：默认的假设
- **备择假设（H1）**：需要验证的假设
- **P值**：在原假设成立的前提下，观察到当前数据或更极端数据的概率
- **显著性水平（α）**：通常取0.05

### 3.4.2 假设检验的流程

1. 提出假设：原假设和备择假设
2. 选择检验方法：根据数据类型选择
3. 计算检验统计量：t值、卡方值等
4. 计算P值
5. 做出判断：P值 < α 则拒绝原假设

### 3.4.3 常见检验方法

| 方法 | 适用场景 | 示例 |
| :--- | :--- | :--- |
| **t检验** | 比较两组均值 | A/B测试中比较两组转化率 |
| **卡方检验** | 比较分类变量的分布 | 检验性别与购买偏好是否相关 |
| **方差分析** | 比较多组均值 | 检验不同广告策略的效果差异 |

## 3.5 案例分析：价格指数的统计分析

### 3.5.1 数据特征

- 数据来源：电商平台的价格数据
- 分析目标：了解价格分布特征

### 3.5.2 分析方法

- 描述性统计：计算均值、中位数、标准差
- 数据可视化：绘制直方图、箱线图
- 假设检验：检验不同平台的价格是否有显著差异

## 3.6 课堂研讨

### 议题一：统计方法选择

**讨论题**：假设你有一组用户行为数据，请分析：
1. 用什么指标描述数据的集中趋势？
2. 用什么图表展示数据的分布？
3. 用什么方法检验两组用户的差异？

### 议题二：统计结果解读

**讨论题**：某电商平台进行A/B测试，结果显示新版本的转化率比旧版本高2%，P值为0.03。请分析：
1. 这个结果是否显著？
2. 是否应该上线新版本？
3. 还需要考虑什么因素？
