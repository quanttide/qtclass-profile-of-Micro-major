# 第2章 指标定义

## 2.1 章节概述

**章节摘要**
本章将介绍指标在数据分析中的核心作用，讲解指标定义的方法与原则。通过本章学习，学生将掌握如何根据业务场景设计合适的指标体系。

---

## 2.2 指标的概念与作用

### 2.2.1 什么是指标

指标是衡量业务状况的数值。它是业务目标的量化表示，用于描述业务现状、评估业务效果、指导业务决策。

### 2.2.2 指标的作用

- **描述业务现状**：用数值展示业务的状态
- **评估业务效果**：衡量业务目标的达成情况
- **指导业务决策**：基于数据做出判断和选择

---

## 2.3 好的指标的标准（SMART原则）

| 原则 | 英文 | 含义 | 示例 |
| :--- | :--- | :--- | :--- |
| **具体** | Specific | 指标要明确 | "用户数"比"用户活跃度"更具体 |
| **可衡量** | Measurable | 结果要可量化 | 销售额、订单数 |
| **可实现** | Achievable | 目标要现实 | 不要设定不切实际的目标 |
| **相关** | Relevant | 与业务相关 | 指标要与业务目标相关 |
| **有时限** | Time-bound | 有明确的时间要求 | 日活、月活、年收入 |

---

## 2.4 指标定义方法

### 2.4.1 北极星指标

**定义**：最能反映业务核心价值的单一指标

**特点**：与业务目标强相关、可衡量、可行动

**示例**：
- 电商：GMV（成交总额）
- 社交：DAU（日活跃用户数）
- 内容：用户时长

### 2.4.2 OSM模型（目标-策略-指标）

| 层次 | 定义 | 示例 |
| :--- | :--- | :--- |
| **目标（Objective）** | 业务目标 | 提升销售额 |
| **策略（Strategy）** | 实现目标的策略 | 提升转化率、提高客单价 |
| **指标（Measure）** | 衡量策略的指标 | 转化率、客单价、复购率 |

### 2.4.3 指标分层体系

- **L1**：公司级指标（北极星指标）
- **L2**：部门级指标
- **L3**：业务线指标
- **L4**：功能级指标

### 2.4.4 指标字典的建立

- 指标名称
- 指标定义
- 计算公式
- 数据来源
- 负责人
- 更新频率

## 2.5 常见业务指标

### 2.5.1 电商类指标

| 指标 | 定义 | 计算公式 |
| :--- | :--- | :--- |
| **GMV** | 成交总额 | 所有订单金额之和 |
| **转化率** | 下单用户占访问用户的比例 | 下单用户数 / 访问用户数 |
| **客单价** | 平均每单金额 | GMV / 订单数 |
| **复购率** | 重复购买用户的比例 | 重复购买用户数 / 总购买用户数 |

### 2.5.2 用户类指标

| 指标 | 定义 | 计算公式 |
| :--- | :--- | :--- |
| **DAU** | 日活跃用户数 | 当日登录或使用产品的用户数 |
| **MAU** | 月活跃用户数 | 当月登录或使用产品的用户数 |
| **留存率** | 用户在一定时间后仍活跃的比例 | 第N天仍活跃的用户数 / 第1天新增用户数 |
| **用户时长** | 用户使用产品的平均时长 | 总使用时长 / 用户数 |
